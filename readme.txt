== Description ==

This Plugin reads out the XML-API of your binlayer.de Account and shows the stats on the Admin¥s Dashboard.
Current version is for WordPress 2.5+ only.

For more infos check out the [Plugin Website](http://www.michaelplas.de/wordpress/binlayer-de-wordpress-dashboard-plugin/ "Plugin Website")!

== Installation ==
Installation is, as usual :

1. Upload `binlayer-dashboard-widget` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Go to your Dashboard
4. Edit the Widget
6. Goto "Tools and API¥s" on binlayer.de after you logged in
7. Copy the Part after "http://binlayerde/api-" into the corresponding Field.
8. If needed, edit the Userlevel for accessing the Widget and deactivate the Button.
